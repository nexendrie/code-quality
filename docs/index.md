Code Quality
============

This package provides code quality tools for Nexendrie Project with default configuration.

Links
-----

Primary repository: https://gitlab.com/nexendrie/code-quality
GitHub repository: https://github.com/nexendrie/code-quality
Packagist: https://packagist.org/packages/nexendrie/code-quality

Installation
------------
The best way to install it is via Composer. Just add **nexendrie/code-quality** to your dev dependencies.
